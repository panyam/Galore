Grammar Syntax

Galore uses a BNF-like syntax for defining grammars. This page covers the complete syntax reference.

Basic Rules

A grammar rule has the form:

NonTerminal -> production1 | production2 | ...

Each production is a sequence of symbols (terminals or non-terminals).

Terminals

Terminals can be:

  • String literals: "keyword" or 'keyword'
  • Regular expressions: /[0-9]+/
  • Named tokens: NUMBER, IDENTIFIER

Example Grammar

Expr -> Expr "+" Term | Term
Term -> Term "*" Factor | Factor
Factor -> "(" Expr ")" | NUMBER

NUMBER -> /[0-9]+/

Operator Precedence

You can define operator precedence and associativity using pragmas:

%left "+" "-"
%left "*" "/"
%right "^"
